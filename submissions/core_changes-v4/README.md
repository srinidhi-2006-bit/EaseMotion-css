# Timeline Component

## What does this do?
Vertical timeline for events, roadmaps, work experience. Alternating and left-aligned layouts with color markers.

## How is it used?
```html
<div class="ease-timeline ease-timeline-alternate">
  <div class="ease-timeline-item">
    <div class="ease-timeline-marker"></div>
    <div class="ease-timeline-content">...
```

## Why is it useful?
Fills missing timeline component with alternating/left layouts, color variants, responsive mobile collapse, dark mode.
