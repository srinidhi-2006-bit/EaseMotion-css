# Animated Text Shadow with Flicker & Glow (`ease-text-shadow`)

A high-impact, intermediate visual text animation utility designed to emulate a retro cyberpunk neon tube aura using multi-layered CSS text shadows and structural voltage flicker keyframe parameters.

## Technical Composition Breakdown

- **Multi-Layered Shimmer Glow Array:** Stacks concentric multi-step `text-shadow` layers moving smoothly from crisp inner whites down to high-radius indigo hues (`#6366f1`).
- **Asymmetric Voltage Flicker Modulation:** Links a dual-animation matrix sequence targeting specific timeline timestamps (e.g., `20%`, `55%`) to dump the shadow text mapping momentarily, producing an organic flicker effect.
- **Vestibular Comfort Guardrails:** Outfitted with an explicit accessibility media query that completely strips out the rapid opacity fluctuations, freezing the element into a continuous soft pulse.

## Workspace Tree Map
- `demo.html`: Independent playground sandbox housing the live neon text layout.
- `style.css`: Clean modular tracking stylesheet configuration blueprints.
