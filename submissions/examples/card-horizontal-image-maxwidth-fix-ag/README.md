# Card Horizontal Image Column Fix (Issue #14076)\n\n## What does this do?\nDemonstrates a correct horizontal card layout where the image column has `flex: 0 0 160px` to constrain its width, leaving room for the body content.\n\n## How is it used?\n```html\n<div class="ease-card ease-card-horizontal">\n  <div class="ease-img-col">...</div>\n  <div class="ease-card-body">...</div>\n</div>\n```\n\n## Why is it useful?\nWithout a constrained flex basis on the image column, the image fills 100% of the card width on desktop, pushing the body text out of view.
