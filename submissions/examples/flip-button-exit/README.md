# Animation: Flip Button Exit

## Description
A premium, lightweight 3D flip-away exit animation effect specifically designed for button components.

## Features
- Pure CSS keyframe animations (Zero JavaScript dependencies)
- 3D perspective depth rotation around the Y-axis
- Responsive and performant (utilizes GPU-accelerated `transform` and `opacity`)
- Complete cross-browser compatibility (Chrome, Firefox, Safari, Edge)

## Usage
Apply the `.exit-active` class dynamically to the button to trigger the exit animation:
```html
<button class="flip-btn exit-active">Click to Exit</button>
```
