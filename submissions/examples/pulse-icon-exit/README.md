# Animation: Pulse Icon Exit

## Description
A delightful exit animation effect for icon components that pulses (scales up slightly) to signal activation, before shrinking down and fading to 0 opacity.

## Features
- Pure CSS keyframe animations (Zero JavaScript dependencies for animation)
- Immersive cubic-bezier scaling pulse
- Responsive and highly performant (GPU-accelerated `transform` and `opacity` transition)
- Cross-browser compatible (Chrome, Firefox, Safari, Edge)

## Usage
Apply the `.exit-active` class dynamically to the icon to trigger the pulse exit animation:
```html
<button class="pulse-icon exit-active">
  <!-- SVG Icon -->
</button>
```
