Features

  • CSS scroll-snap — smooth, native scrolling
  • Autoplay — configurable via data-carousel-autoplay attribute (ms)
  • Dot navigation — click any dot to jump to that slide
  • Prev/Next buttons — keyboard and click navigation
  • Pause on hover — autoplay stops when hovering
  • ARIA accessible — carousel, slide, and tab roles
  • Dark mode — compatible via CSS variables

Usage

<div class="ease-carousel" data-carousel-autoplay="4000">
  <div class="ease-carousel-track">
    <div class="ease-carousel-slide">...</div>
    <div class="ease-carousel-slide">...</div>
  </div>
  <button class="ease-carousel-btn ease-carousel-prev">&#10094;</button>
  <button class="ease-carousel-btn ease-carousel-next">&#10095;</button>
  <div class="ease-carousel-dots">
    <button class="ease-carousel-dot active" data-index="0"></button>
    <button class="ease-carousel-dot" data-index="1"></button>
  </div>
</div>
